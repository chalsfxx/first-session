CSS Selectors

Los selectores CSS se dirigen y seleccionan dlos elementos HTML a los que se quiere dar estilo.

En concreto, los selectores CSS permiten seleccionar varios elementos a la vez.

Son útiles cuando quieres aplicar los mismos estilos a más de un elemento HTML, porque no te repetirás escribiendo las mismas líneas de código para diferentes elementos.

Los selectores de CSS también son útiles cuando quieres hacer un cambio: sólo tienes que hacer el cambio en un lugar, lo que te ahorra mucho tiempo.

Simple CSS Selectors

Los selectores le permiten seleccionar partes específicas de su documento con fines de estilización.

Los selectores simples seleccionan directamente uno o varios elementos:

  • Utilizando el selector universal, *.
  • Basado en el nombre/tipo del elemento.
  • Basado en el valor de la clase del elemento.
  • Basado en el valor del ID del elemento.

By learning how the most simple selectors work, you can understand how to use the more complex ones.

The simple selectors will most often be the ones you will use the most and the ones you will be the most familiar with if you have some experience writing CSS code.
